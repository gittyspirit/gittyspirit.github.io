Installing Hexo Word Counter

发表于2023-08-14|更新于2024-05-05
|字数总计:198|阅读时长:1分钟|阅读量:

Step 1 Make sure that you have installed Rust

If you haven’t installed Rust yet, follow the commands below.

MacOS,Linux or other Unix-like OS

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Windows

Please download this file first. https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe

And also install Visual C++
https://visualstudio.microsoft.com/thank-you-downloading-visual-studio/?sku=Community&channel=Release&version=VS2022&source=VSFeaturesPage&passive=false&cid=2030

Then run rustup-init.exe to install Rust

Step 2 Install Hexo-word-counter

npm install hexo-word-counter --save
hexo clean

Here is the code in _config.xml

path: yourblog/_config.yml

symbols_count_time:
  symbols: true
  time: true
  total_symbols: true
  total_time: true
  exclude_codeblock: false
  wpm: 275
  suffix: "mins."

For NeXT Theme

path: yourblog/themes/next/_config.yml

post_meta:
  item_text: true
  created_at: true
  updated_at:
    enable: true
    another_day: true
  categories: true

symbols_count_time:
  separated_meta: true
  item_text_total: false

Step 3 Save the file and Deploy your site

hexo clean && hexo g && hexo d
